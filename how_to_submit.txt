Copy the flag from it's location, it will be a long set of random numbers/letters.. Make sure you have the whole thing, no breaks or spaces.

Run flag.sh like so:

./flag.sh cbbe3ec55dd050e749918770af0a40b6d8192679

This should ask for your passphrase if you have one. NOTE: YOU MUST DO THIS WITH THE GPG KEY YOU SUBMITTED TO THE CTF. Any other key won't count for your team.
